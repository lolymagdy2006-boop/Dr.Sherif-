Using Local Data Storage

Local data storage allows web applications to store data directly inside the user's browser instead of requesting it repeatedly from a server.

Faster Loading

Less Server Strain

Practical Example


// Saving data to localStorage
localStorage.setItem("theme", "dark");

// Retrieving data
const currentTheme = localStorage.getItem("theme");
console.log(currentTheme); // Outputs: dark

Quick Check

Question: Which tool would you use to save a user's theme preference so it persists even after they restart their computer?

Click to reveal the answer
localStorage. Unlike sessionStorage, data in localStorage has no expiration date and stays saved until explicitly removed via code or by clearing the browser cache.
