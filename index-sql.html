Storage Layer

Database

A database is an organized collection of structured information stored electronically. It allows for efficient data management, enabling users to store, retrieve, and update information seamlessly.

Query Language

SQL

Structured Query Language (SQL) is the standard programming language for relational databases. It is used to perform tasks such as updating data or retrieving specific data sets through queries.

SELECT name FROM users WHERE role = 'admin';
03
Core Components

Entity

An entity is a real-world object or concept that can be uniquely identified (e.g., a Person, Place, or Object). In a relational database, an entity is typically represented as a table.

04
Connectivity

Relationship

Relationships define how entities are connected to one another. They establish the logic between tables, such as how a 'Customer' entity relates to an 'Order' entity.
